# 🔮 Agent 04: Predictive Risk Analytics Agent

## Overview
The **Predictive Risk Analytics Agent** uses a combination of historical EHR data, real-time vital trends, and demographic factors to predict clinical deterioration, readmission probability, and complication risks. 

It provides transparent explanations for its predictions using simulated **SHAP (Shapley Additive Explanations)** values and a detailed **SBAR (Situation, Background, Assessment, Recommendation)** narrative generated by TinyLlama.

---

## 🏗️ Design & Logic
The agent operates at the intersection of **ML Transparency** and **Clinical Narrative**:

1.  **Risk Modeling**:
    *   Simulates a gradient-boosted ML model (like XGBoost) to compute risk percentages.
    *   Evaluates features like age, multi-morbidity (e.g., Diabetes + Hypertension), and acute vital trends (e.g., dropping SpO2).
2.  **Explainability (SHAP)**:
    *   Identifies the specific "Risk Factors" that contributed most to the score.
    *   Provides clinical descriptions for why these features increased the risk.
3.  **SBAR Communication**:
    *   TinyLlama synthesizes the Situation, Background, Assessment, and Recommendation (SBAR) to prepare the physician for immediate intervention.

---

## 🔄 Workflow
1.  **Ingestion**: Receives a `PatientProfile` (History + Trends + Labs).
2.  **Risk Calculation**: Executes the predictive logic to determine the `Risk Tier`.
3.  **Explainability**: Extracts top contributing factors.
4.  **Narrative Generation**: TinyLlama builds a structured clinical summary.
5.  **Intervention Suggestions**: Maps the risk level and factors to specific clinical interventions (e.g., RRT consult).
6.  **Output**: Returns a `RiskAssessment` object.

---

## 🛠️ Tech Stack
*   **Predictive Framework**: Python (Scikit-learn/XGBoost simulation)
*   **Narrative Engine**: TinyLlama-1.1B
*   **Web Framework**: FastAPI
*   **Data Validation**: Pydantic v2

## 🚀 Getting Started

### 1. Running the Agent
Start the microservice on port `8004`:
```bash
export PYTHONPATH=$PYTHONPATH:.
python3 -m agents.agent_04_risk.main
```

### 2. Testing & Endpoint Calling

**Option A: Automated Test Script**
```bash
python3 -m agents.agent_04_risk.test_agent
```

**Option B: Manual Curl Call**
```bash
curl -X POST http://localhost:8004/predict \
     -H "Content-Type: application/json" \
     -d '{
          "patient_id": "PAT-RISK-001",
          "age": 82,
          "comorbidities": ["Diabetes", "Hypertension"],
          "recent_vital_trends": {"SpO2": "Decreasing"},
          "last_lab_results": {"Glucose": 240},
          "medications": ["Metformin"]
         }'
```
